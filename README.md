# Leafetch
<h3>A very simple system fetch written in Nim.</h3>

# To build
### 0) Install Nim(https://nim-lang.org/)

### 1) Clone the repo
```fish
git clone https://github.com/kuraria/leafetch.git
```
### 2) cd into the repo
```fish
cd leafetch
```

### 3) Build with `nimble`
```fish
nimble build
```
